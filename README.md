Objective

The objective of Day 2 was to implement a lead capture mechanism that allows users to submit potential customer information through a web form. The submitted information is validated and stored in the database for further processing.

This component acts as the data collection layer of the Lead Generation System.

Technologies Used

Django – Web framework used for backend logic

Django ModelForm – For automatic form generation

HTML Templates – User interface for lead submission
